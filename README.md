# Youtube-video-search-and-downloader
A python script made for searching for youtube videos then downloading, the app will allow you to search up a video you want to watch and download it into the repository you are working on the project with, if this isn't long enough then im out of ideas
## ON LINUX
1 - git clone https://github.com/Goldcrescent/Youtube-video-search-and-downloader.git                                                                                                                                           
2 - cd Youtube-video-search-and-downloader
3 - pip install httpx
4 - pip install yt-dlp
5 - python/python3 main.py
